## Overview

SwarmFix operates on a **pay-on-acceptance** model. You only pay after reviewing and accepting the delivered pull request. No upfront costs, no subscriptions, no hidden fees. If you're not satisfied, reject the PR and owe nothing.

## Pricing Tiers

### 🆓 Free Fix (Phase 1)

- **Price:** $0
- **Scope:** Simple bug fixes, typo corrections, small improvements
- **Turnaround:** Best-effort
- **Purpose:** Validate SwarmFix capability on real-world codebases
- **Limitations:** No SLA, limited availability

### ⚡ Quick Fix ($10–$50)

- **Price:** Starting at $10
- **Scope:** Straightforward bug fixes, small feature additions, configuration changes
- **Typical complexity:** Single-file or few-file changes
- **Turnaround:** ~24–48 hours
- **Examples:**
  - Fix a crash or runtime error
  - Add input validation
  - Update dependencies
  - Correct broken CI configuration

### 🔧 Complex Fix ($50–$500)

- **Price:** $50–$500 (quoted per issue)
- **Scope:** Multi-file refactors, feature implementation, complex bug resolution
- **Typical complexity:** Cross-module changes, architectural improvements
- **Turnaround:** 2–7 days
- **Examples:**
  - Implement a new API endpoint
  - Refactor authentication flow
  - Resolve concurrency bugs
  - Add comprehensive test coverage for a module

## Pricing Modifiers

- **Urgency premium** — Expedited turnaround is available for time-sensitive issues at an additional cost.
- **Codebase complexity** — Large or unusual codebases may incur a surcharge due to increased analysis time.
- **Custom quote** — Issues that fall outside standard tiers will receive a custom price quote during triage.

## Payment Information

- **Accepted currencies:** USDC and ETH on supported networks
- **Model:** Pay-on-acceptance — you only pay after reviewing and accepting the PR
- **No upfront costs, no subscription**
- **Payment instructions** are provided after PR acceptance
- **Payment due** within 7 days of acceptance
